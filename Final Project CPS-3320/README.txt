Data and map representation of accidents in new york city on 2020. This is a jupyter notebook. csv database and ny .png image need to be downloaded and the paths for all the code sections need to be updated to your local path. To view the code working just install the required libraries and execute the jupyter notebook in order one section at the time. since the data file is large the program might take a few seconds to generate maps and output depending on your computer.

Required libraries and extensions for jupyter notebook (Run on terminal)

Pip install pandas
Pip install IPython
Pip install matplotlib
Pip install numpy
Pip install geopandas
Pip install os
pip install ipywidgets
pip install keplergl
install --py --sys-prefix keplergl
jupyter nbextension enable keplergl --py --sys-prefix
jupyter nbextension enable --py widgetsnbextension
